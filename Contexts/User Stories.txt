User — Who Lost Item
User Story
- I want to register/login so I can access the app and manage my reports.
- I want to fill out my personal information form so the staff can verify my identity when I claim.
- I want to search for my lost item to see if it has already been reported or found.
- I want to report my lost item with details (date, location, description, photo).
- I want to update/edit my lost item report if I entered wrong details.
- I want to track the status of my lost item report (pending, found, claimed).
- I want to upload proof of ownership when I request a claim.
- I want to receive notifications when staff update or match my lost item.
- I want to schedule a pickup date after my claim is approved.
- I want to message staff if I have concerns about my lost item.
- I want to cancel a claim request if I realize I no longer need it.

User — Who Found Item
User Story
- I want to register/login so I can report a found item.
- I want to fill out my personal information form so staff can contact me if needed.
- I want to report a found item with details (description, location, photo, time found).
- I want to hand over the item to the staff through a scheduled meeting or drop-off.
- I want to update the details of a found item report if I made a mistake.
- I want to message staff if I see an item on the app that I know about.
- I want to remain anonymous when reporting a found item (optional, for privacy).
- I want to track the status of the item I reported to know if it has been claimed.

Staff
User Story
- I want to register/login so I can manage lost and found items.
- I want to fill out my staff information form so the system knows my role.
- I want to review lost and found item reports submitted by users.
- I want to post the reported lost/found items on the system.
- I want to verify proof of ownership before approving a claim.
- I want to accept or reject claim requests depending on validity.
- I want to schedule item pickup after approving a claim.
- I want to keep records of all items received and claimed for tracking.
- I want to message users if I need clarification about a report or claim.
- I want to post verified found items into the system so they're visible to users.
